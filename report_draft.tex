\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
This report presents an internship project on plant image classification using convolutional neural networks. The main goal is to compare a custom CNN trained from scratch with pretrained ResNet50-based transfer learning approaches under a shared experimental pipeline. The project currently uses a merged dataset built from PlantVillage and Leafsnap, resulting in 76 label groups and 51{,}504 images. The report focuses on dataset preparation, model design, training strategy, evaluation metrics, and comparative analysis of performance, efficiency, and generalization.

\chapter{Introduction}
\section{Background}
Plant image classification is an important computer vision task with applications in agriculture, biodiversity monitoring, and intelligent advisory systems. Deep learning models, especially convolutional neural networks, have demonstrated strong performance in visual recognition problems.

\section{Problem Statement}
This internship investigates how different CNN training strategies perform on plant classification. The main comparison is between:
\begin{itemize}
    \item a custom CNN trained from scratch,
    \item ResNet50 with feature extraction,
    \item ResNet50 with fine-tuning.
\end{itemize}

\section{Objectives}
The objectives of this project are:
\begin{itemize}
    \item to build a reproducible training pipeline for plant classification,
    \item to compare scratch-trained and pretrained CNN models,
    \item to evaluate accuracy, macro F1-score, and training efficiency,
    \item to produce a usable showcase model for inference.
\end{itemize}

\section{Scope}
The current implementation focuses on a 76-class merged dataset:
\begin{itemize}
    \item PlantVillage images grouped at crop level: Pepper, Potato, Tomato,
    \item Leafsnap images grouped at genus level.
\end{itemize}

\chapter{Dataset Preparation}
\section{Source Datasets}
\subsection{PlantVillage}
The local PlantVillage subset used in this project contains 15 folders and 20{,}639 images, limited to Pepper, Potato, and Tomato categories.

\subsection{Leafsnap}
Leafsnap contributes both lab and field images. The dataset contains 185 species in the lab split and 184 species in the field split.

\section{Cleaning and Restructuring Pipeline}
The dataset preparation process consists of:
\begin{enumerate}
    \item extracting and cleaning raw filenames,
    \item restructuring PlantVillage by crop name,
    \item restructuring Leafsnap by genus name,
    \item removing metadata files and keeping image files only.
\end{enumerate}

\section{Final Dataset}
The final dataset used for training is \texttt{dataset\_plant\_classification}:
\begin{itemize}
    \item 76 classes,
    \item 51{,}504 images,
    \item image-only contents (\texttt{.jpg}, \texttt{.jpeg}, \texttt{.png}).
\end{itemize}

\chapter{Model Design}
\section{Custom CNN Baseline}
The custom CNN uses four convolutional blocks with channel sizes:
\[
3 \rightarrow 32 \rightarrow 64 \rightarrow 128 \rightarrow 256
\]
Each block includes convolution, batch normalization, ReLU activation, and pooling. The classifier head uses dropout and a final linear layer.

\section{ResNet50 Feature Extraction}
In the feature extraction setting, the pretrained ResNet50 backbone is frozen and only the classifier head is trained.

\section{ResNet50 Fine-Tuning}
In the fine-tuning setting, the final residual block and classifier head are updated during training.

\chapter{Limitations and Future Work}
\section{Current Limitations}
\begin{itemize}
    \item The dataset is a merged plant classification dataset rather than a pure disease-classification benchmark.
    \item Class imbalance remains significant, especially for Tomato and some Leafsnap genera.
    \item Long-running Kaggle sessions may interrupt full experiments.
\end{itemize}

\section{Future Work}
\begin{itemize}
    \item Resume-capable training checkpoints,
    \item weighted loss or class-balancing strategies,
    \item comparison with additional backbones such as MobileNetV2 or EfficientNet,
    \item dedicated inference demo or deployment interface.
\end{itemize}

\chapter{Conclusion}
This internship project establishes a practical pipeline for large-scale plant classification using both scratch-trained and pretrained CNN models. The report is expected to show the relative strengths of custom CNNs and transfer learning approaches in terms of performance, training efficiency, and deployment readiness.
